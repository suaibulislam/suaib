# Content Templates

This folder contains templates for adding new content to the website.

## Adding a Blog Post

1. Create a new `.md` file in `src/content/blog/`
2. Name it with a slug (e.g., `my-first-post.md`)
3. Copy the frontmatter from `_blog-template.md`
4. Write your content below the `---`

## Adding a Project

1. Create a new `.md` file in `src/content/projects/`
2. Name it with a slug (e.g., `cool-thing.md`)
3. Copy the frontmatter from `_project-template.md`
4. Write your content below the `---`

That's it. Astro picks them up automatically.
