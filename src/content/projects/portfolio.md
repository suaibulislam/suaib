# this very website

this portfolio is a labor of love that started as a vanilla html/css project and was recently migrated to **astro**. the goal was to keep the "sketchbook" charm while gaining the performance and maintainability of a modern framework.

## why astro?

- **speed:** it ships zero javascript to the browser by default. all the heavy lifting happens at build time.
- **content collections:** i can manage my "rants" and projects as simple markdown files (like the one you're reading).
- **component-based:** everything is broken down into small, reusable chunks, making it way easier to update the design without breaking the site.

## preserved features

even though it's a "modern" stack, i made sure to keep the soul of the original site:
- **the particle system:** the subtle background animation still follows your mouse.
- **timezone widget:** so you know exactly how late it is in bangladesh when i'm coding.
- **sketchbook aesthetics:** the wobbly borders, dotted lines, and hand-drawn feel are all preserved and now managed through a cleaner css system.

## technical details

- **stack:** astro, typescript (for better code safety), and vanilla css.
- **deployment:** statically generated and hosted on vercel for instant global loading.
- **seo:** built-in sitemaps and structured data so google actually knows who i am.
